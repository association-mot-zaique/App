# Contribuer à MOT-ZAIQUE

Merci de votre intérêt pour le projet MOT-ZAIQUE.

MOT-ZAIQUE est une application de Communication Alternative et Améliorée (CAA) développée par l'association MOT-ZAIQUE.

Les contributions sont les bienvenues afin d'améliorer l'application et de la rendre plus utile aux personnes utilisant la CAA.

---

## Types de contributions

Vous pouvez contribuer de plusieurs manières :

- signaler des bugs
- proposer des améliorations
- améliorer la documentation
- proposer des traductions
- contribuer au développement
- partager des retours d'utilisation

---

## Rejoindre le projet

Pour les développeurs et testeurs les plus motivés, vous pouvez devenir bénévole auprès de l'association MOT-ZAIQUE en nous contactant à l'adresse suivante :
contact@motzaique.fr

---


## Signaler un problème

Avant de créer un problème (issue), merci de vérifier qu'il n'existe pas déjà.

Lorsque vous signalez un bug, indiquez :

- la version de l'application
- votre système (Android, version...)
- les étapes permettant de reproduire le problème
- le comportement attendu

---

## Proposer une modification

Pour contribuer au code :

1. Forker le dépôt
2. Créer une branche dédiée :
	feature/[ma-nouvelle-fonctionnalite]
	fix/correction-[du-bug]

3. Effectuer vos modifications
4. Créer une Pull Request

---

## Qualité du code

Merci de :

- garder un code lisible
- documenter les modifications importantes
- éviter les changements sans rapport avec l'objectif de la contribution

---

## Licence des contributions

En proposant une contribution au projet MOT-ZAIQUE, vous acceptez que celle-ci soit distribuée sous la licence GNU General Public License v3.0.

---

## Respect des utilisateurs

MOT-ZAIQUE est un projet destiné à améliorer l'accessibilité et la communication.

Les contributions doivent respecter :

- les utilisateurs de CAA
- les personnes en situation de handicap
- les familles et professionnels accompagnants

---

Merci de contribuer à un outil libre au service de la communication.
