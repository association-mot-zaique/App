# mot-zaique

App Flutter para comunicacion asistida con pictogramas de ARASAAC.

## Funciones

- Busqueda de pictogramas en ARASAAC.
- Construccion de frases tocando pictos.
- Reproduccion de frase con Text-to-Speech.
- Favoritos persistentes protegidos por PIN.
- Interfaz pastel responsive para moviles y tablets.

## Ejecutar

```bash
flutter pub get
flutter run
```
