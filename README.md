# MOT-ZAIQUE

Application de Communication Alternative et Améliorée (CAA) destinée à faciliter la communication des personnes non verbales grâce à l'utilisation de pictogrammes.

MOT-ZAIQUE est un projet porté par l'association MOT-ZAIQUE.

Site :
https://motzaique.fr

Contact :
contact@motzaique.fr

---

## Présentation

MOT-ZAIQUE permet aux utilisateurs de créer et utiliser des supports de communication à partir de pictogrammes.

L'application est conçue pour fonctionner principalement hors ligne.

Une connexion internet est nécessaire uniquement pour rechercher et importer des pictogrammes depuis des sources externes compatibles, notamment ARASAAC.

---

## Fonctionnalités

- Création de tableaux de communication
- Utilisation de pictogrammes
- Personnalisation des supports de communication
- Fonctionnement hors connexion
- Protection des paramètres par code PIN

---

## Public concerné

MOT-ZAIQUE est une application de Communication Alternative et Améliorée (CAA).

Elle s'adresse notamment aux personnes ayant besoin d'un support de communication basé sur des pictogrammes.
